#  Event Poster Generator

A free, easy-to-use tool for creating professional event posters. Perfect for community organizers, churches, schools, and small businesses.

## Features

- 4 beautiful template styles (Modern, Bold, Minimal, Retro)
- Customizable event details
- Instant preview
- High-resolution download (800x1200px)
- 100% free and open source

## How to Use

1. Visit the live site: [Event Poster Generator](#)
2. Choose your template style
3. Fill in your event details
4. Click "Generate Poster"
5. Download your poster!

## Built With

- HTML5 Canvas
- Vanilla JavaScript
- CSS3

## Impact

Helping communities create professional event materials without needing design skills or expensive software.
